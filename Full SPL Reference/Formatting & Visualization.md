_The "Presentation" phase: Preparing data for the user._

| **Command**     | **Description**                                                  | **Example** |
| --------------- | ---------------------------------------------------------------- | ----------- |
| **`table`**     | Retains data in tabular format, keeping only specified fields. | `           |
| **`chart`**     | Matrix/Pivot output. Limit: 2 dimensions (`over` and `by`).    | `           |
| **`gauge`**     | Formats a single number for a speedometer visualization.         | `           |
